## Installation

### Git Clone
```
git clone 
```
### Open Folder
```
cd nordvpn_switcher
```
### Run
```
python3 nordvpn_switcher.py
```

## Usage

<b> Enter the Seconds to Reconnect to an NordVPN Connection! </b>

## Recommend!

<b>The time have to be more than 30 Seconds (Otherwise NordVPN have connection issues!)</b>
